# [IT 협조 요청] 타 사업장 간 내부 업무 플랫폼 접속 허용 요청

---

**요청 부서:** 생산기술본부

**요청일:** 2026년 3월 10일

**요청자:** (이름 기입)

---

## 1. 요청 개요

생산기술본부에서 개발 중인 내부 업무 플랫폼(웹 애플리케이션)을 타 사업장에서도 접속할 수 있도록 네트워크 방화벽 포트 개방을 요청드립니다.

## 2. 현재 상태

- **서버 위치:** 향남공장 사내망
- **서버 IP:** 10.12.1.237
- **서비스 포트:** TCP 3000 ~ 3001
- **접속 주소:** http://10.12.1.237:3000
- **현재 상태:** 향남공장 내부망(10.12.0.0/22)에서만 접속 가능

## 3. 요청 사항

타 사업장 네트워크에서 아래 목적지로 접근할 수 있도록 방화벽 포트 개방 또는 라우팅 허용을 요청드립니다.

| 항목 | 내용 |
| --- | --- |
| 출발지 | 평택 사업장 (10.13.1.148) |
| 목적지 | 향남공장 (10.12.1.237) |
| 프로토콜/포트 | TCP / 3000 ~ 3001 |
| 방향 | 양방향 (평택 ↔ 향남) |
| 용도 | 생산기술본부 내부 업무 플랫폼 접속 |
| 희망 기간 | 임시 (테스트 기간 동안, 약 ___개월) |

## 4. 용도 설명

해당 플랫폼은 생산기술본부 내부 업무용 웹 애플리케이션으로, 문서 관리, 결재, 설비 관리, 안전 점검 등의 기능을 포함하고 있습니다. 평택 사업장 소속 직원들도 동일한 시스템에 접속하여 업무를 처리할 수 있도록 하기 위한 요청입니다.

## 5. 접속 방법

방화벽 개방 후, 평택 사업장 PC에서 웹 브라우저에 아래 주소를 입력하여 접속합니다.

```
http://10.12.1.237:3000
```

## 6. 추가 요청 사항 (선택)

- 현재 DHCP로 IP를 할당받고 있어 IP가 변경될 수 있습니다. 가능하다면 **고정 IP 할당**을 함께 요청드립니다.
- 향후 정식 운영 시 **내부 DNS 등록**(예: prod-platform.사내도메인)도 검토 부탁드립니다.

## 7. 보안 참고

- 외부(인터넷) 개방이 아닌 **사내망 간 통신만** 요청합니다.
- 로그인 인증이 적용되어 있으며, 인가된 사용자만 접속 가능합니다.
